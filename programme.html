APS Programme

Before start (6:00–6:30pm)

  • Welcome people as they arrive
  • Ask if they’d like to do a talk → add their name to the app
  • Count names in the app → decide if a break is needed
  • 5mins before start → Remind to grab a drink if they like

Opening: App overview (6:30pm)

Explain how the event will run with the app.

Run through the app’s demo mode so people see the topics, timer and controls in action.

Opening: Reminders (6:40pm)

  • Everyone is welcome to provide feedback
  • This is a safe and supportive environment
  • Speaking is optional
  • If you’re not ready to speak, feel free to just enjoy the vibe of the group.

Group Introductions (6:45) ~ keep short & sweet

  • Name
  • Start with an easy warm-up question (make your own or check with Bat/Glen)
→ Impromptu Talks Begin (6:45–7ish)

Half-way Break (if needed, around ~7:30)

  • Give people a chance to order a drink / toilet break
  • For late-comers → Now is a good time to add name to the app
→ 2nd Half Impromptu Talks (7:45ish)

Wrap up (roughly 8:30pm)

  • (If there is time) Open the floor for any reflections/lessons learned
  • (If host would like) Everyone is welcome to give feedback for the host
  • Congratulate everyone for showing up 🙌
  • Whatsapp QR codes available to scan on the table
  • Interested in hosting? → chat with Bat/Glen
  • Final Announcements from Bat/Glen, if any
